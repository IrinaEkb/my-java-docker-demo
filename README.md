# My Java Docker Demo

A simple Java 21 project running inside Docker.

## How to run

Clone the repository:

```bash
git clone https://github.com/IrinaEkb/my-java-docker-demo.git
cd my-java-docker-demo
Start the Docker container:
docker-compose up --build
Output:
Hello from Docker Java!
Running Tests
To run the JUnit tests, use Maven:
mvn test
GitHub Actions
GitHub Actions will automatically run tests and build Docker images on every push to this repository.
